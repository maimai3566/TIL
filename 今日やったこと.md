# 11/2
- Djangoのテンプレートに関すること
    - HTMLの中では、pythonからの戻り値は{}の中に記述
    - {{}}はデータを格納・・・{{ abc }}のように波かっこと変数の間にスペースを入れないとエラーになるかもしれない（まだ未検証）
    - {% for xx in xxxx %}~{% endfor %}でループでデータを取得し、テーブルの<li>に値を格納
    - {%%}はfor文やif文が使える
    - {% url %}はDjangoのURLパターンに基づき、指定されたビューに対するURLを作る
